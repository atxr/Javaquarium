I : Probleme

	2 especes, conditions initial connues, une proie, une predateur
	Proie reproduise de facon "linéaire" et se font manger
	Predacteur meurent de faim de facon "lineaire" et se reproduisent si ils sont nourrient

II : Forme des equations: 
	
	X(t+dt) - X(t) = r.X(t) - m.Y(t) pour les proies 
	Y(t+dt) - Y(t) = -f.Y(t) + n.X(t) pour les predateurs


Puis 3 especes, +1 en bout de chaine, étude 1ere en fonction 3eme
